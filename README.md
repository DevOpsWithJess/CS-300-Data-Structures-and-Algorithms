# C++ Data Structures and Algorithms - Course Planner

Author: Jessica Johnson
Language: C++

---

## Overview

This project implements a console-based course planner using core data structures in C++. The application reads course information from a CSV file and allows users to view, search, and organize courses and their prerequisites.

The goal of the project is to demonstrate how different data structures can be used to efficiently store, retrieve, and organize real-world data.

---

## Technologies Used

C++
Standard Template Library (STL)
File I/O
CSV Data Processing

---

## Program Features

• Load course data from a CSV file
• Display a full list of available courses
• Search for a specific course
• Display prerequisite information
• Organize course data efficiently using data structures
• Interactive console-based menu

---

## Concepts Demonstrated

• Hash tables for fast key-based lookup
• Vectors for dynamic data storage
• File input/output for reading datasets
• Sorting and searching algorithms
• Modular program structure

---

## Running the Program

1. Compile the C++ program
2. Ensure the dataset file is present:

```
ABCU_Advising_Program_Input.csv
```

3. Run the executable

Example:

```
g++ CoursePlanner.cpp -o planner
./planner
```
