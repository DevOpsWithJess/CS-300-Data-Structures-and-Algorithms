# C++ Data Structures and Algorithms - Course Planner

Author: Jessica Johnson
Language: C++

---

## Overview

This C++ application loads course data from a CSV file into a hash table and allows users to interact with the data through a console-based menu. The program allows users to view courses, search for a specific course, and display prerequisite information.

The project demonstrates how data structures can be used to efficiently store and retrieve structured data.

---

## Technologies Used

- C++
- Standard Template Library (STL)
- File I/O
- CSV Data Processing

---

## Program Features

- Load course data from a CSV file
- Display a full list of available courses
- Search for a specific course
- Display prerequisite information
- Organize course data efficiently using data structures
- Interactive console-based menu

---

## Concepts Demonstrated

- Hash tables for fast key-based lookup
- Vectors for dynamic data storage
- File input/output for reading datasets
- Sorting and searching algorithms
- Menu-based user interaction 
- Modular program structure

---

## Running the Program

1. Compile the C++ program
2. Ensure the dataset file is present:

```
ABCU_Advising_Program_Input.csv
```

3. Run the executable

Example:

```
g++ CoursePlanner.cpp -o planner
./planner
```
