# SNHU CS 300 Final Project- Project Two
**ABC University Course Planner**

This C++ program loads course data from a CSV file into a hash table. This allows users to:
* View all courses in Alphanumeric order
* Search for a specific course
* Display any prerequisites

##Primary Concepts
* Hash Tables
* Vectors for Sorting Data from the Hash Tables
* User interaction with Menu
